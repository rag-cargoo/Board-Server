- 각 단계에서 추가된 주요 기능, 관련 이슈 링크, 테스트 방법 등을 기록하세요.
- 새로운 실습 브랜치를 만들었다면 위 표에 행을 추가하십시오.
- 단계별로 필요한 초기화 명령이나 실행 순서가 있다면 별도 섹션을 만들어 정리하세요.
